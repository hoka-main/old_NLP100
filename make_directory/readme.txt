		-------使用方法-------

はじめに「sample.csv」を「ディレクトリリスト.csv」に名前を変更してから「make_directory.exe」を実行し、問題なく動作することを確認してください。
※動作しない場合はmix_spまでご連絡ください。




「make_directory」の利用には、利用者が.csvファイルを用意する必要があります。

.csvファイルは利用者の「メモ帳」などのテキストエディタまたは「Excel」を使って作成することができます。

Excelでディレクトリリスト.csvを作成する場合、ファイル名を

	「性,名」や「コードNo,店舗名」

等、A列とB列に２列で分割して作成すると正常に動作します。

ただし、文字コードによっては文字化けする可能性がありますので、その場合はメモ帳を使用して作成してください。
またはアルファベットで作成してください。



「make_directory.exe」は.csvファイルが「ディレクトリリスト.csv」という名前でのみ動作します。

最初から入っている「ディレクトリリスト」に上書き保存するか、編集してご利用ください。

またmake_directory.exeはディレクトリリスト.csvの内容が次の形式でのみ正常に機能します。



＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

	○○,○○　→　○○_○○

	○○,　　　→　○○_

	,○○　　　→　_○○

	,	　 →　_

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊


最初から入っているsample.csvは次の形式を網羅しております。

・ヘッダーが存在する。
・同じディレクトリ名を指定している。
・ディレクトリ名の1列目、または2列目が未指定である。
・ディレクトリ名に３列目以降が存在する。


１列に想定している「,」の数が２つ以上ある場合は２つ目以降の文字列が削除されたフォルダが作成されます。

ディレクトリリスト.csvをテキストエディタで開き、作成されたディレクトリを参照しつつ動作を確認してください。




		-------想定されるエラー-------



make_directory.exeはディレクトリリスト.csvのデータに「,」がない文字列が存在する場合、エラーが発生し動作を停止します。

その場合はディレクトリリスト.csvをテキストエディタで開き、該当文字列に「,」を記入してください。
または該当文字列を削除してください。



make_directory.exeはディレクトリリスト.csvのデータに「' '」（スペースなどの空白）が存在する場合、エラーが発生し動作を停止します。

その場合は「' '」を削除してください。



make_directory.exeはディレクトリリスト.csvのデータに「???」などの文字化けが存在する場合、エラーが発生し動作を停止します。

エラーが出た場合はディレクトリリスト.csvをテキストエディタで開き、文字コードを確認してください。
文字コードの変更方法がわからない場合はディレクトリリスト.csvをテキストエディタで新規作成し、拡張子を「.csv」にすることを忘れずに保存してください。
または、アルファベットで作成し直し、文字化けした文字列を削除してください。



保存形式を「○○_○○」以外で保存したい場合はmix_spにご連絡いただき、オプションを購入してください。
